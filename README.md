# Simple Task Manager

## Objective
A simple web application to manage daily tasks.

## Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Add tasks using Enter key

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project files.
2. Open `index.html` in any browser.
